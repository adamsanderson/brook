# Brook

Brook is a simple feed reader built to work with Chrome and Firefox.

## Development

To start developing, run:

```
yarn install
yarn dev # The plugin will build and launch Firefox by default

# Or choose a browser
yarn dev:firefox
yarn dev:chrome
```

To see debugging messages in Firefox, use the `Browser Console`.  To debug visit `about:debugging`.

To access special debugging menus, shift click the primary menu in the plugin.
